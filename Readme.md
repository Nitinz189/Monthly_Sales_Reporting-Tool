# 🚀 Monthly Sales Reporting Automation Tool (Excel + VBA)

## 📌 Overview
This project is a **fully automated Monthly Sales Reporting Tool** built using **Microsoft Excel (VBA)**.  
It simulates a real-world **Reporting Analyst workflow**, where raw sales data is transformed into structured, navigable, and decision-ready reports.

The tool eliminates repetitive manual work and enables **instant report generation with navigation and summaries**.

---

## 🎯 Problem Statement
In many organizations, reporting analysts:
- Spend hours manually creating reports
- Navigate between multiple sheets
- Build repetitive structures every month
- Risk human errors in linking and formatting

👉 This project solves that by **automating the entire reporting layer inside Excel**.

## ⚙️ Features

### ✅ Automated Table of Contents (TOC)
- Dynamically generates clickable hyperlinks to all worksheets
- Provides quick navigation across reports
- Displays sheet-level summaries (from cell A1)

### ✅ One-Click Report Structuring
- Converts raw sheets into a structured reporting format
- Eliminates manual linking between sheets

### ✅ Hyperlinked Navigation System
- Each report section is interconnected
- Enables fast movement across dashboards and data

### ✅ Scalable Design
- Works with any number of sheets
- Easily adaptable for:
  - Monthly reports
  - Department-wise reports
  - Client reporting
